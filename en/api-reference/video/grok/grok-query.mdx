# Grok Query Video Task

Use the task ID to query the status and results of Grok video generation tasks.

## Authentication

All requests require a Bearer token in the request header:

```bash
Authorization: Bearer {{key}}
```

## Path Parameters

<ParamField path="id" type="string" required>
  Unique identifier of the video generation task, returned from the create task endpoint

  Example: `video_fd35ee52-2a98-44a6-b930-29a88ce9b8fd`
</ParamField>

## Response Parameters

<ResponseField name="id" type="string">
  Unique identifier of the video generation task
</ResponseField>

<ResponseField name="object" type="string">
  Object type, fixed as `video`
</ResponseField>

<ResponseField name="created" type="integer">
  Task creation timestamp
</ResponseField>

<ResponseField name="model" type="string">
  Model name used (grok_video3 or grok_video3_pro)
</ResponseField>

<ResponseField name="status" type="string">
  Task status:
  - `queued` - In queue
  - `processing` - Processing
  - `completed` - Completed
  - `failed` - Failed
</ResponseField>

<ResponseField name="video_url" type="string">
  Generated video URL, returned when status is `completed`
</ResponseField>

<ResponseField name="actualDuration" type="number">
  Actual output video duration (seconds), returned when status is `completed`
</ResponseField>

<ResponseField name="error" type="string">
  Error message, returned when status is `failed`
</ResponseField>

## Status Flow

<Steps>
  <Step title="queued">
    Task has been created and added to the queue, waiting to be processed
  </Step>
  <Step title="processing">
    Task is being processed, Grok model is generating the video
  </Step>
  <Step title="completed">
    Task completed, video generated successfully, results available via `video_url`
  </Step>
</Steps>

## Polling Recommendations

<Info>
**Polling Strategy:**
- Recommended polling interval: 30-60 seconds
- Grok generation speed varies by model and duration:
  - grok_video3 (6-10s): Usually completes in 2-4 minutes
  - grok_video3 (20-30s): Usually completes in 4-8 minutes
  - grok_video3_pro (10s): Usually completes in 3-6 minutes
- Avoid overly frequent requests to prevent rate limiting
- Stop polling when status is `completed` or `failed`
</Info>

## Video URL Notes

<Warning>
**Important:**
- Generated video URLs are typically valid for 24 hours
- Please download and save videos to your storage system promptly
- Videos will be inaccessible after the URL expires
- Video files are typically in MP4 format
</Warning>

```python Python (Polling Example)
import requests
import time

def wait_for_video(task_id, api_key, max_wait=600, interval=30):
    """
    Poll and wait for video generation to complete

    Args:
        task_id: Task ID
        api_key: API key
        max_wait: Maximum wait time (seconds)
        interval: Polling interval (seconds)

    Returns:
        Video URL or None
    """
    url = f'https://zcbservice.aizfw.cn/kyyReactApiServer/v1/result/{task_id}'
    headers = {'Authorization': f'Bearer {api_key}'}

    start_time = time.time()

    while time.time() - start_time < max_wait:
        response = requests.get(url, headers=headers)
        data = response.json()

        status = data.get('status')

        if status == 'completed':
            print(f"Video generated successfully!")
            print(f"Video duration: {data.get('actualDuration')} seconds")
            return data.get('video_url')
        elif status == 'failed':
            print(f"Generation failed: {data.get('error')}")
            return None

        print(f"Current status: {status}, waiting...")

        time.sleep(interval)

    print("Wait timeout")
    return None

# Usage example
video_url = wait_for_video('video_fd35ee52-2a98-44a6-b930-29a88ce9b8fd', 'your_api_key')
if video_url:
    print(f"Video URL: {video_url}")
```

// Poll and wait for video generation to complete
async function waitForVideo(taskId, apiKey, maxWait = 600000, interval = 30000) {
  const startTime = Date.now();

  while (Date.now() - startTime < maxWait) {
    const response = await fetch(
      `https://zcbservice.aizfw.cn/kyyReactApiServer/v1/result/${taskId}`,
      {
        headers: { 'Authorization': `Bearer ${apiKey}` }
      }
    );

    const data = await response.json();
    const status = data.status;

    if (status === 'completed') {
      console.log('Video generated successfully!');
      console.log(`Video duration: ${data.actualDuration} seconds`);
      return data.video_url;
    } else if (status === 'failed') {
      console.log(`Generation failed: ${data.error}`);
      return null;
    }

    console.log(`Current status: ${status}, waiting...`);
    await new Promise(resolve => setTimeout(resolve, interval));
  }

  console.log('Wait timeout');
  return null;
}

// Usage example
const videoUrl = await waitForVideo('video_fd35ee52-2a98-44a6-b930-29a88ce9b8fd', 'your_api_key');
if (videoUrl) {
  console.log(`Video URL: ${videoUrl}`);
}
