# KyyVideo2 Asset List

Query paginated asset list for the current user's asset group with support for filtering by resource type and status.

## Authentication

All requests require a Bearer token in the request header:

```bash
Authorization: Bearer {{key}}
```

## Request Parameters

<ParamField path="page" type="integer">
  Current page number, default 1
</ParamField>

<ParamField path="pageSize" type="integer">
  Number of items per page, default 10
</ParamField>

<ParamField path="status" type="array">
  Asset status filter array

  Available options:
  - Empty array `[]` - Query all statuses
  - `["Active"]` - Asset uploaded successfully, ready to use
  - `["Processing"]` - Asset is being processed
  - `["Failed"]` - Asset upload failed
  - `["Active", "Processing"]` - Query multiple statuses
</ParamField>

<ParamField path="assetType" type="array">
  Asset type filter array

  Available options:
  - Empty array `[]` - Query all types
  - `["Image"]` - Query images only
  - `["Video"]` - Query videos only
  - `["Audio"]` - Query audio only
  - `["Document"]` - Query documents only
  - `["Image", "Video"]` - Query multiple types
</ParamField>

## Response Parameters

<ResponseField name="code" type="integer">
  Response status code
  - `0` - Success
  - Other values - Failure
</ResponseField>

<ResponseField name="msg" type="string">
  Response message, `null` on success, error message on failure
</ResponseField>

<ResponseField name="data" type="object">
  Paginated data object

  <Expandable title="data object properties">
    <ResponseField name="records" type="array">
      Asset records array

      <Expandable title="records array element properties">
        <ResponseField name="assetId" type="string">
          Asset ID, format: `asset-{timestamp}-{random}`
        </ResponseField>

        <ResponseField name="name" type="string">
          Asset name
        </ResponseField>

        <ResponseField name="assetType" type="string">
          Asset type: Image, Video, Audio, Document
        </ResponseField>

        <ResponseField name="url" type="string">
          Asset URL address
        </ResponseField>

        <ResponseField name="status" type="string">
          Asset status
          - `Active` - Asset uploaded successfully, ready to use
          - `Processing` - Asset is being processed
          - `Failed` - Asset upload failed
        </ResponseField>

        <ResponseField name="createTime" type="string">
          Creation time, format: `YYYY-MM-DD HH:mm:ss`
        </ResponseField>
      </Expandable>
    </ResponseField>

    <ResponseField name="total" type="integer">
      Total number of records
    </ResponseField>

    <ResponseField name="size" type="integer">
      Number of items per page
    </ResponseField>

    <ResponseField name="current" type="integer">
      Current page number
    </ResponseField>

    <ResponseField name="pages" type="integer">
      Total number of pages
    </ResponseField>
  </Expandable>
</ResponseField>

<Info>
**Pagination Notes:**
- Default display is 10 records per page
- Calculate total pages using `total` and `pages`
- Use `current` to know the current page number
- Adjust `pageSize` based on actual needs
</Info>

<Tip>
**Best Practices:**
- Regularly query asset list to understand uploaded assets
- Use type filtering to quickly find needed assets
- Save `assetId` of frequently used assets to avoid duplicate uploads
- Use `createTime` to understand asset upload time
</Tip>

```python Python
import requests

# Query all assets
response = requests.post(
    'https://zcbservice.aizfw.cn/kyyAiPower/kyyVideo2/asset/list',
    headers={
        'Authorization': 'Bearer {{key}}',
        'Content-Type': 'application/json'
    },
    json={
        'page': 1,
        'pageSize': 10,
        'status': [],
        'assetType': []
    }
)

result = response.json()
assets = result['data']['records']

print(f"Total {result['data']['total']} assets")
for asset in assets:
    print(f"Asset ID: {asset['assetId']}, Name: {asset['name']}, Type: {asset['assetType']}")

# Query image assets
image_response = requests.post(
    'https://zcbservice.aizfw.cn/kyyAiPower/kyyVideo2/asset/list',

        'Content-Type': 'application/json'
    },
    json={
        'page': 1,
        'pageSize': 20,
        'status': [],
        'assetType': ['Image']
    }
)
```
