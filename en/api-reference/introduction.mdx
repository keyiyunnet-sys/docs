## SDKs & Code Examples

Official SDK support for Python, Node.js, and Java. All SDKs are OpenAI-compatible, requiring only base URL configuration changes for seamless migration.

<CodeGroup>
```python Python SDK
from openai import OpenAI

client = OpenAI(
    base_url="https://api.globalaiopc.com/v1",
    api_key="your-apimart-key"
)
```

```javascript Node.js SDK
import OpenAI from 'openai';

const client = new OpenAI({
  baseURL: 'https://api.globalaiopc.com/v1',
  apiKey: 'your-apimart-key'
});
```

```java Java SDK
import com.openai.OpenAI;

OpenAI client = OpenAI.builder()
    .baseUrl("https://api.globalaiopc.com/v1")
    .apiKey("your-apimart-key")
    .build();
```
</CodeGroup>
