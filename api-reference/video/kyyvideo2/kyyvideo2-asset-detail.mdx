# KyyVideo2 素材详情

获取单个素材的详细信息，包括素材名称、类型、状态、URL 等。

## 认证

所有请求都需要在请求头中包含 Bearer token：

```bash
Authorization: Bearer {{key}}
```

## 路径参数

<ParamField path="assetId" type="string" required>
  素材 ID，从上传素材接口或素材列表接口获取

  示例：`asset-20260330142158-q75m8`
</ParamField>

## 响应参数

<ResponseField name="code" type="integer">
  响应状态码
  - `0` - 成功
  - 其他值 - 失败
</ResponseField>

<ResponseField name="msg" type="string">
  响应消息，成功时为 `null`，失败时返回错误信息
</ResponseField>

<ResponseField name="data" type="object">
  素材详细信息对象

  <Expandable title="data 对象属性">
    <ResponseField name="assetId" type="string">
      素材 ID，格式：`asset-{timestamp}-{random}`
    </ResponseField>

    <ResponseField name="name" type="string">
      素材名称
    </ResponseField>

    <ResponseField name="assetType" type="string">
      素材类型：Image、Video、Audio、Document
    </ResponseField>

    <ResponseField name="url" type="string">
      素材 URL 地址
    </ResponseField>

    <ResponseField name="status" type="string">
      素材状态
      - `Active` - 素材上传成功，可正常使用
      - `Processing` - 素材处理中
      - `Failed` - 素材上传失败
    </ResponseField>

    <ResponseField name="createTime" type="string">
      创建时间，格式：`YYYY-MM-DD HH:mm:ss`
    </ResponseField>
  </Expandable>
</ResponseField>

<Info>
**使用建议：**
- 上传素材后，可以通过此接口查询素材处理状态
- 在使用素材前，建议先查询确认状态为 `Active`
- 可以通过此接口获取素材的最新 URL
</Info>

<Tip>
**最佳实践：**
- 素材上传后，轮询此接口直到状态变为 `Active`
- 保存素材 ID，避免重复上传相同素材
- 定期检查素材状态，确保素材可用
</Tip>

```python Python
import requests

# 查询素材详情
asset_id = "asset-20260330142158-q75m8"
response = requests.get(
    f'https://zcbservice.aizfw.cn/kyyReactApiServer/kyyVideo2/asset/{asset_id}',
    headers={'Authorization': 'Bearer {{key}}'}
)

result = response.json()
if result['code'] == 0:
    asset = result['data']
    print(f"素材名称: {asset['name']}")
    print(f"素材类型: {asset['assetType']}")
    print(f"素材状态: {asset['status']}")
    print(f"素材URL: {asset['url']}")
else:
    print(f"查询失败: {result['msg']}")

# 轮询等待素材处理完成
import time

def wait_for_asset_ready(asset_id, max_wait=300, interval=5):
    """等待素材处理完成"""
    start_time = time.time()

    while time.time() - start_time < max_wait:
        response = requests.get(
            f'https://zcbservice.aizfw.cn/kyyReactApiServer/kyyVideo2/asset/{asset_id}',
            headers={'Authorization': 'Bearer {{key}}'}
        )

        result = response.json()
        if result['code'] == 0:
            status = result['data']['status']
            print(f"当前状态: {status}")

            if status == 'Active':
                print("素材已就绪！")
                return result['data']
            elif status == 'Failed':
                print("素材处理失败")
                return None

        time.sleep(interval)

    print("等待超时")
    return None

# 使用示例
asset_data = wait_for_asset_ready("asset-20260330142158-q75m8")
```

```javascript JavaScript
// 轮询等待素材处理完成
async function waitForAssetReady(assetId, maxWait = 300000, interval = 5000) {
  const startTime = Date.now();

  while (Date.now() - startTime < maxWait) {
    const response = await fetch(
      `https://zcbservice.aizfw.cn/kyyReactApiServer/kyyVideo2/asset/${assetId}`,
      {
        headers: { 'Authorization': 'Bearer {{key}}' }
      }
    );

    const result = await response.json();
    if (result.code === 0) {
      const status = result.data.status;
      console.log(`当前状态: ${status}`);

      if (status === 'Active') {
        console.log('素材已就绪！');
        return result.data;
      } else if (status === 'Failed') {
        console.log('素材处理失败');
        return null;
      }
    }

    await new Promise(resolve => setTimeout(resolve, interval));
  }

  console.log('等待超时');
  return null;
}
```
