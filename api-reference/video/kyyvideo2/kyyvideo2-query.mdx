# KyyVideo2 查询视频任务

使用任务 ID 查询 KyyVideo2 视频生成任务的状态和结果。

## 认证

所有请求都需要在请求头中包含 Bearer token：

```bash
Authorization: Bearer {{key}}
```

## 路径参数

<ParamField path="id" type="string" required>
  视频生成任务的唯一标识符，从创建任务接口返回

  示例：`video_fd35ee52-2a98-44a6-b930-29a88ce9b8fd`
</ParamField>

## 响应参数

<ResponseField name="id" type="string">
  视频生成任务的唯一标识符
</ResponseField>

<ResponseField name="object" type="string">
  对象类型，固定为 `video`
</ResponseField>

<ResponseField name="created" type="integer">
  任务创建时间戳
</ResponseField>

<ResponseField name="model" type="string">
  使用的模型名称（kyyvideo2、kyyvideo2_fast、kyyvideo2_pro、kyyvideo2_fast_pro）
</ResponseField>

<ResponseField name="status" type="string">
  任务状态：
  - `queued` - 排队中
  - `processing` - 处理中
  - `completed` - 已完成
  - `failed` - 失败
</ResponseField>

<ResponseField name="video_url" type="string">
  生成的视频URL，当 status 为 `completed` 时返回
</ResponseField>

<ResponseField name="error" type="string">
  错误信息，当 status 为 `failed` 时返回
</ResponseField>

## 状态流程

<Steps>
  <Step title="queued">
    任务已创建并加入队列，等待处理
  </Step>
  <Step title="processing">
    任务正在处理中，KyyVideo2 模型正在生成视频
  </Step>
  <Step title="completed">
    任务完成，视频生成成功，可通过 `video_url` 获取结果
  </Step>
</Steps>

## 轮询建议

<Info>
**轮询策略：**
- 建议轮询间隔：20-40 秒
- 快速模型（kyyvideo2_fast、kyyvideo2_fast_pro）生成速度更快，可适当缩短轮询间隔
- 避免过于频繁的请求以免触发限流
- 当状态为 `completed` 或 `failed` 时停止轮询
</Info>

## 视频URL说明

<Warning>
**重要提示：**
- 生成的视频URL有效期为 1 天
- 请及时下载并保存视频到你的存储系统
- URL 过期后将无法访问视频
- 视频文件通常为 MP4 格式
- 根据 `generateAudio` 参数，视频可能包含或不包含音频
</Warning>

## 生成时间预估

<CardGroup cols={2}>
  <Card title="标准模型" icon="clock">
    kyyvideo2 / kyyvideo2_pro

    预计生成时间：2-5 分钟
  </Card>
  <Card title="快速模型" icon="bolt">
    kyyvideo2_fast / kyyvideo2_fast_pro

    预计生成时间：1-3 分钟
  </Card>
</CardGroup>

<Note>
**影响生成时间的因素：**
- 视频时长：时长越长，生成时间越长
- 模型类型：快速模型生成速度更快
- 参考素材：使用多模态参考（图片、视频、音频）可能增加处理时间
- 系统负载：高峰期可能需要更长的排队时间
</Note>

## 模型生成速度对比

| 模型 | 平均生成时间 | 适用场景 |
|------|------------|---------|
| kyyvideo2 | 2-5 分钟 | 标准质量，支持参考视频 |
| kyyvideo2_fast | 1-3 分钟 | 快速生成，支持参考视频 |
| kyyvideo2_pro | 3-6 分钟 | 高质量输出 |
| kyyvideo2_fast_pro | 2-4 分钟 | 高质量快速生成 |

```python Python
import requests
import time

# 创建任务
create_response = requests.post(
    'https://zcbservice.aizfw.cn/kyyReactApiServer/v1/kyyvideo2/videos',
    headers={
        'Authorization': 'Bearer {{key}}',
        'Content-Type': 'application/json'
    },
    json={
        'model': 'kyyvideo2_fast',
        'prompt': '一只猫在雨中跳舞，电影风格',
        'duration': 10,
        'aspect_ratio': '16:9'
    }
)

task_id = create_response.json()['id']
print(f"任务已创建，ID: {task_id}")

# 轮询查询任务状态
while True:
    query_response = requests.get(
        f'https://zcbservice.aizfw.cn/kyyReactApiServer/v1/result/{task_id}',
        headers={'Authorization': 'Bearer {{key}}'}
    )

    result = query_response.json()
    status = result['status']

    print(f"当前状态: {status}")

    if status == 'completed':
        print(f"视频生成成功！")
        print(f"视频URL: {result['video_url']}")
        break
    elif status == 'failed':
        print(f"视频生成失败: {result['error']}")
        break

    # 等待 30 秒后再次查询
    time.sleep(30)
```

```javascript JavaScript
// 创建任务
const createResponse = await fetch('https://zcbservice.aizfw.cn/kyyReactApiServer/v1/kyyvideo2/videos', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer {{key}}',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    model: 'kyyvideo2_fast',
    prompt: '一只猫在雨中跳舞，电影风格',
    duration: 10,
    aspect_ratio: '16:9'
  })
});

const { id: taskId } = await createResponse.json();
console.log(`任务已创建，ID: ${taskId}`);

// 轮询查询任务状态
async function pollTaskStatus(taskId) {
  while (true) {
    const queryResponse = await fetch(
      `https://zcbservice.aizfw.cn/kyyReactApiServer/v1/result/${taskId}`,
      {
        headers: { 'Authorization': 'Bearer {{key}}' }
      }
    );

    const result = await queryResponse.json();
    console.log(`当前状态: ${result.status}`);

    if (result.status === 'completed') {
      console.log('视频生成成功！');
      console.log(`视频URL: ${result.video_url}`);
      return result;
    } else if (result.status === 'failed') {
      console.log(`视频生成失败: ${result.error}`);
      throw new Error(result.error);
    }

    // 等待 30 秒后再次查询
    await new Promise(resolve => setTimeout(resolve, 30000));
  }
}

await pollTaskStatus(taskId);
```
