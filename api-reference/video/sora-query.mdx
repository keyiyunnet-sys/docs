# Sora 查询视频任务

使用任务 ID 查询 Sora 视频生成任务的状态和结果。

## 认证

所有请求都需要在请求头中包含 Bearer token：

```bash
Authorization: Bearer {{key}}
```

## 路径参数

<ParamField path="id" type="string" required>
  视频生成任务的唯一标识符，从创建任务接口返回
</ParamField>

## 响应参数

<ResponseField name="id" type="string">
  视频生成任务的唯一标识符
</ResponseField>

<ResponseField name="object" type="string">
  对象类型，固定为 `video.generation`
</ResponseField>

<ResponseField name="created" type="integer">
  任务创建时间戳
</ResponseField>

<ResponseField name="model" type="string">
  使用的模型名称
</ResponseField>

<ResponseField name="status" type="string">
  任务状态：
  - `queued` - 排队中
  - `processing` - 处理中
  - `completed` - 已完成
  - `failed` - 失败
</ResponseField>

<ResponseField name="video_url" type="string">
  生成的视频URL，当 status 为 `completed` 时返回
</ResponseField>

<ResponseField name="error" type="string">
  错误信息，当 status 为 `failed` 时返回
</ResponseField>

## 状态流程

<Steps>
  <Step title="queued">
    任务已创建并加入队列，等待处理
  </Step>
  <Step title="processing">
    任务正在处理中，Sora 模型正在生成视频
  </Step>
  <Step title="completed">
    任务完成，视频生成成功，可通过 `video_url` 获取结果
  </Step>
</Steps>

## 轮询建议

<Info>
**轮询策略：**
- 建议轮询间隔：30-60 秒（视频生成时间较长）
- 避免过于频繁的请求以免触发限流
- 当状态为 `completed` 或 `failed` 时停止轮询
</Info>

## 视频URL说明

<Warning>
**重要提示：**
- 生成的视频URL有效期通常为 24 小时
- 请及时下载并保存视频到你的存储系统
- URL 过期后将无法访问视频
- 视频文件通常为 MP4 格式
</Warning>
