# KyyVideo2 素材列表

分页查询当前用户素材组下的素材列表，支持按资源类型、状态进行筛选。

## 认证

所有请求都需要在请求头中包含 Bearer token：

```bash
Authorization: Bearer {{key}}
```

## 请求参数

<ParamField path="page" type="integer">
  当前页码，默认 1
</ParamField>

<ParamField path="pageSize" type="integer">
  每页数量，默认 10
</ParamField>

<ParamField path="status" type="array">
  素材状态筛选数组

  可选值：
  - 空数组 `[]` - 查询所有状态
  - `["Active"]` - 素材上传成功，可正常使用
  - `["Processing"]` - 素材处理中
  - `["Failed"]` - 素材上传失败
  - `["Active", "Processing"]` - 查询多种状态
</ParamField>

<ParamField path="assetType" type="array">
  素材类型筛选数组

  可选值：
  - 空数组 `[]` - 查询所有类型
  - `["Image"]` - 仅查询图片
  - `["Video"]` - 仅查询视频
  - `["Audio"]` - 仅查询音频
  - `["Document"]` - 仅查询文档
  - `["Image", "Video"]` - 查询多种类型
</ParamField>

## 响应参数

<ResponseField name="code" type="integer">
  响应状态码
  - `0` - 成功
  - 其他值 - 失败
</ResponseField>

<ResponseField name="msg" type="string">
  响应消息，成功时为 `null`，失败时返回错误信息
</ResponseField>

<ResponseField name="data" type="object">
  分页数据对象

  <Expandable title="data 对象属性">
    <ResponseField name="records" type="array">
      素材记录数组

      <Expandable title="records 数组元素属性">
        <ResponseField name="assetId" type="string">
          素材 ID，格式：`asset-{timestamp}-{random}`
        </ResponseField>

        <ResponseField name="name" type="string">
          素材名称
        </ResponseField>

        <ResponseField name="assetType" type="string">
          素材类型：Image、Video、Audio、Document
        </ResponseField>

        <ResponseField name="url" type="string">
          素材 URL 地址
        </ResponseField>

        <ResponseField name="status" type="string">
          素材状态
          - `Active` - 素材上传成功，可正常使用
          - `Processing` - 素材处理中
          - `Failed` - 素材上传失败
        </ResponseField>

        <ResponseField name="createTime" type="string">
          创建时间，格式：`YYYY-MM-DD HH:mm:ss`
        </ResponseField>
      </Expandable>
    </ResponseField>

    <ResponseField name="total" type="integer">
      总记录数
    </ResponseField>

    <ResponseField name="size" type="integer">
      每页数量
    </ResponseField>

    <ResponseField name="current" type="integer">
      当前页码
    </ResponseField>

    <ResponseField name="pages" type="integer">
      总页数
    </ResponseField>
  </Expandable>
</ResponseField>

## 使用场景

<CardGroup cols={2}>
  <Card title="查询所有素材" icon="list">
    不设置任何筛选条件，获取所有素材列表
  </Card>
  <Card title="按类型筛选" icon="filter">
    通过 assetType 参数筛选特定类型的素材
  </Card>
  <Card title="按状态筛选" icon="filter-circle-dollar">
    通过 status 参数筛选特定状态的素材
  </Card>
  <Card title="分页浏览" icon="book">
    使用 page 和 pageSize 参数进行分页查询
  </Card>

<Info>
**分页说明：**
- 默认每页显示 10 条记录
- 通过 `total` 和 `pages` 可以计算总页数
- 使用 `current` 可以知道当前所在页码
- 建议根据实际需求调整 `pageSize`
</Info>

<Tip>
**最佳实践：**
- 定期查询素材列表，了解已上传的素材
- 使用类型筛选快速找到需要的素材
- 保存常用素材的 `assetId`，避免重复上传
- 通过 `createTime` 了解素材上传时间
</Tip>

```python Python
import requests

# 查询所有素材
response = requests.post(
    'https://zcbservice.aizfw.cn/kyyReactApiServer/kyyVideo2/asset/list',
    headers={
        'Authorization': 'Bearer {{key}}',
        'Content-Type': 'application/json'
    },
    json={
        'page': 1,
        'pageSize': 10,
        'status': [],
        'assetType': []
    }
)

result = response.json()
assets = result['data']['records']

print(f"总共 {result['data']['total']} 个素材")
for asset in assets:
    print(f"素材ID: {asset['assetId']}, 名称: {asset['name']}, 类型: {asset['assetType']}")

# 查询图片素材
image_response = requests.post(
    'https://zcbservice.aizfw.cn/kyyReactApiServer/kyyVideo2/asset/list',
    headers={
        'Authorization': 'Bearer {{key}}',
        'Content-Type': 'application/json'
    },
    json={
        'page': 1,
        'pageSize': 20,
        'status': [],
        'assetType': ['Image']
    }
)
```

import java.net.http.HttpClient;

import java.net.http.HttpResponse;

import java.util.Map;
import java.util.List;

HttpClient client = HttpClient.newHttpClient();
ObjectMapper mapper = new ObjectMapper();

Map<String, Object> requestBody = Map.of(
    "page", 1,
    "pageSize", 10,
    "status", List.of(),
    "assetType", List.of()
);

    .header("Content-Type", "application/json")
    .POST(HttpRequest.BodyPublishers.ofString(mapper.writeValueAsString(requestBody)))
    .build();

HttpResponse<String> response = client.send(request, HttpResponse.BodyHandlers.ofString());
System.out.println(response.body());
