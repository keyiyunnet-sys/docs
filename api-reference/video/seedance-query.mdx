# Seedance 查询视频任务

使用任务 ID 查询 Seedance 视频生成任务的状态和结果。

## 认证

所有请求都需要在请求头中包含 Bearer token：

```bash
Authorization: Bearer {{YOUR_API_KEY}}
```

## 路径参数

<ParamField path="id" type="string" required>
  视频生成任务的唯一标识符，从创建任务接口返回
</ParamField>

## 响应参数

<ResponseField name="id" type="string">
  视频生成任务的唯一标识符
</ResponseField>

<ResponseField name="object" type="string">
  对象类型，固定为 `video`
</ResponseField>

<ResponseField name="created" type="integer">
  任务创建时间戳
</ResponseField>

<ResponseField name="model" type="string">
  使用的模型名称
</ResponseField>

<ResponseField name="status" type="string">
  任务状态：
  - `queued` - 排队中
  - `processing` - 处理中
  - `completed` - 已完成
  - `failed` - 失败
  - `cancelled` - 已取消
</ResponseField>

<ResponseField name="video_url" type="string">
  生成的视频 URL，当 status 为 `completed` 时返回
</ResponseField>

<ResponseField name="error" type="string">
  错误信息，当 status 为 `failed` 时返回
</ResponseField>

## 状态流程

<Steps>
  <Step title="queued">
    任务已创建并加入队列，等待处理
  </Step>
  <Step title="processing">
    任务正在处理中，Seedance 模型正在生成视频
  </Step>
  <Step title="completed">
    任务完成，视频生成成功，可通过 `video_url` 获取结果
  </Step>
</Steps>

## 轮询建议

<Info>
**轮询策略：**
- 建议轮询间隔：
  - 样片模式：10-20 秒
  - 正式视频：30-60 秒
  - 离线服务（flex）：60-120 秒
- 避免过于频繁的请求以免触发限流
- 当状态为 `completed`、`failed` 或 `cancelled` 时停止轮询
</Info>

## 视频URL说明

<Warning>
**重要提示：**
- 任务 ID 和生成的视频保存 7 天
- 请及时下载并保存视频到你的存储系统
- 7 天后任务记录和视频将被删除
- 视频文件通常为 MP4 格式
</Warning>

```python Python (轮询示例)
import requests
import time

def wait_for_video(task_id, api_key, max_wait=600, interval=30):
    """
    轮询等待视频生成完成
    
    Args:
        task_id: 任务ID
        api_key: API密钥
        max_wait: 最大等待时间（秒）
        interval: 轮询间隔（秒）
    
    Returns:
        视频URL或None
    """
    url = f'https://zcbservice.aizfw.cn/kyyAiPower/v1/result/{task_id}'
    headers = {'Authorization': f'Bearer {api_key}'}
    
    start_time = time.time()
    
    while time.time() - start_time < max_wait:
        response = requests.get(url, headers=headers)
        data = response.json()
        
        status = data.get('status')
        
        if status == 'completed':
            return data.get('video_url')
        elif status == 'failed':
            print(f"生成失败: {data.get('error')}")
            return None
        elif status == 'cancelled':
            print("任务已取消")
            return None
        
        print(f"当前状态: {status}，等待中...")
        
        time.sleep(interval)
    
    print("等待超时")
    return None

# 使用示例
video_url = wait_for_video('video_abc123def456', 'your_api_key')
if video_url:
    print(f"视频生成成功: {video_url}")
```
