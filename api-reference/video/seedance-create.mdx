# Seedance 创建视频任务

使用火山引擎 Seedance（即梦）AI 模型创建高质量视频生成任务，支持文生视频、图生视频（首帧/首尾帧）等多种生成方式。

## 认证

所有请求都需要在请求头中包含 Bearer token：

```bash
Authorization: Bearer {{YOUR_API_KEY}}
```

## 请求参数

<ParamField path="model" type="string" required>
    模型名称，见上方支持的模型列表
</ParamField>

<ParamField path="content" type="array" required>
    输入内容数组，支持文本、图片格式。详见下方 content 数组结构说明
</ParamField>

<ParamField path="ratio" type="string">
    视频宽高比

    可选值：`16:9`、`4:3`、`1:1`、`3:4`、`9:16`、`21:9`、`adaptive`

    <Note>
        默认值：
        - 文生视频：`16:9`（Seedance 1.5 Pro 为 `adaptive`）
        - 图生视频：`adaptive`
    </Note>
</ParamField>

<ParamField path="duration" type="integer">
    视频时长（秒），支持 2-12 秒，默认为 5

    <Note>
        Seedance 1.5 pro 特殊说明：
        - 指定具体时长：4-12 秒的整数
        - 自动选择时长：设置为 `-1`，模型自动选择 4-12 秒
    </Note>
</ParamField>

## Content 数组结构

### 文本内容对象

<ParamField path="type" type="string" required>
    固定值：`text`
</ParamField>

<ParamField path="text" type="string" required>
    文本提示词，中文不超过 500 字，英文不超过 1000 词
</ParamField>

**示例：**
```json
{
  "type": "text",
  "text": "小猫对着镜头打哈欠"
}
```

### 图片内容对象

<ParamField path="type" type="string" required>
    固定值：`image_url`
</ParamField>

<ParamField path="image_url.url" type="string" required>
    图片 URL
</ParamField>

<ParamField path="role" type="string">
    图片角色，根据使用场景选择：
    - `first_frame` 或不填 - 首帧图片（图生视频-首帧）
    - `last_frame` - 尾帧图片（图生视频-首尾帧）
    - `reference_image` - 参考图片（Seedance 1.0 lite i2v，1-4 张）
</ParamField>

**图片要求：**
- 格式：jpeg、png、webp、bmp、tiff、gif（Seedance 1.5 pro 支持 heic、heif）
- 宽高比：(0.4, 2.5)
- 尺寸：(300px, 6000px)
- 大小：< 30 MB

**示例：**
```json
{
  "type": "image_url",
  "image_url": {
    "url": "https://example.com/image.jpg"
  },
  "role": "first_frame"
}
```

## 响应参数

<ResponseField name="id" type="string">
    视频生成任务 ID，保存 7 天，用于后续查询任务状态
</ResponseField>

<ResponseField name="object" type="string">
    对象类型，固定为 `video`
</ResponseField>

<ResponseField name="created" type="integer">
    任务创建时间戳
</ResponseField>

<ResponseField name="model" type="string">
    使用的模型名称
</ResponseField>

<ResponseField name="status" type="string">
    任务状态：
    - `queued` - 排队中
    - `processing` - 处理中
    - `completed` - 已完成
    - `failed` - 失败
    - `cancelled` - 已取消
</ResponseField>

## 注意事项

<Warning>
    **重要限制：**
    - 图片场景互斥：首帧、首尾帧、参考图三种场景不可混用
    - 有声视频功能仅支持 Seedance 1.5 pro 系列
    - 任务 ID 保存 7 天，请及时保存生成的视频
</Warning>

<Info>
    **最佳实践：**
    1. 视频生成为异步处理，需要通过返回的 `id` 查询任务状态和结果
    2. 选择合适的分辨率和时长以平衡质量和成本
    3. 图生视频时，图片质量直接影响生成效果
</Info>

    import java.net.http.HttpClient;
    import java.net.http.HttpRequest;

    .header("Content-Type", "application/json")
    .POST(HttpRequest.BodyPublishers.ofString(mapper.writeValueAsString(requestBody)))
    .build();

    HttpResponse<String> response = client.send(request, HttpResponse.BodyHandlers.ofString());
