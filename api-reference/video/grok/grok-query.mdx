# Grok 查询视频任务

使用任务 ID 查询 Grok 视频生成任务的状态和结果。

## 认证

所有请求都需要在请求头中包含 Bearer token：

```bash
Authorization: Bearer {{key}}
```

## 路径参数

<ParamField path="id" type="string" required>
  视频生成任务的唯一标识符，从创建任务接口返回

  示例：`video_fd35ee52-2a98-44a6-b930-29a88ce9b8fd`
</ParamField>

## 响应参数

<ResponseField name="id" type="string">
  视频生成任务的唯一标识符
</ResponseField>

<ResponseField name="object" type="string">
  对象类型，固定为 `video`
</ResponseField>

<ResponseField name="created" type="integer">
  任务创建时间戳
</ResponseField>

<ResponseField name="model" type="string">
  使用的模型名称（grok_video3 或 grok_video3_pro）
</ResponseField>

<ResponseField name="status" type="string">
  任务状态：
  - `queued` - 排队中
  - `processing` - 处理中
  - `completed` - 已完成
  - `failed` - 失败
</ResponseField>

<ResponseField name="video_url" type="string">
  生成的视频URL，当 status 为 `completed` 时返回
</ResponseField>

<ResponseField name="actualDuration" type="number">
  视频实际输出时长（秒），当 status 为 `completed` 时返回
</ResponseField>

<ResponseField name="error" type="string">
  错误信息，当 status 为 `failed` 时返回
</ResponseField>

## 状态流程

<Steps>
  <Step title="queued">
    任务已创建并加入队列，等待处理
  </Step>
  <Step title="processing">
    任务正在处理中，Grok 模型正在生成视频
  </Step>
  <Step title="completed">
    任务完成，视频生成成功，可通过 `video_url` 获取结果
  </Step>
</Steps>

## 轮询建议

<Info>
**轮询策略：**
- 建议轮询间隔：30-60 秒
- Grok 生成速度因模型和时长而异：
  - grok_video3（6-10秒）：通常 2-4 分钟完成
  - grok_video3（20-30秒）：通常 4-8 分钟完成
  - grok_video3_pro（10秒）：通常 3-6 分钟完成
- 避免过于频繁的请求以免触发限流
- 当状态为 `completed` 或 `failed` 时停止轮询
</Info>

## 视频URL说明

<Warning>
**重要提示：**
- 生成的视频URL有效期通常为 24 小时
- 请及时下载并保存视频到你的存储系统
- URL 过期后将无法访问视频
- 视频文件通常为 MP4 格式
</Warning>

<Note>
**影响生成时间的因素：**
- 视频时长：时长越长，生成时间越长
- 分辨率选择：720p 比 480p 需要更长时间
- 参考图片数量：使用多张参考图片可能增加处理时间
- 系统负载：高峰期可能需要更长的排队时间
</Note>

```python Python (轮询示例)
import requests
import time

def wait_for_video(task_id, api_key, max_wait=600, interval=30):
    """
    轮询等待视频生成完成

    Args:
        task_id: 任务ID
        api_key: API密钥
        max_wait: 最大等待时间（秒）
        interval: 轮询间隔（秒）

    Returns:
        视频URL或None
    """
    url = f'https://zcbservice.aizfw.cn/kyyReactApiServer/v1/result/{task_id}'
    headers = {'Authorization': f'Bearer {api_key}'}

    start_time = time.time()

    while time.time() - start_time < max_wait:
        response = requests.get(url, headers=headers)
        data = response.json()

        status = data.get('status')

        if status == 'completed':
            print(f"视频生成成功！")
            print(f"视频时长: {data.get('actualDuration')} 秒")
            return data.get('video_url')
        elif status == 'failed':
            print(f"生成失败: {data.get('error')}")
            return None

        print(f"当前状态: {status}，等待中...")

        time.sleep(interval)

    print("等待超时")
    return None

# 使用示例
video_url = wait_for_video('video_fd35ee52-2a98-44a6-b930-29a88ce9b8fd', 'your_api_key')
if video_url:
    print(f"视频URL: {video_url}")
```

// 轮询等待视频生成完成
async function waitForVideo(taskId, apiKey, maxWait = 600000, interval = 30000) {
  const startTime = Date.now();

  while (Date.now() - startTime < maxWait) {
    const response = await fetch(
      `https://zcbservice.aizfw.cn/kyyReactApiServer/v1/result/${taskId}`,
      {
        headers: { 'Authorization': `Bearer ${apiKey}` }
      }
    );

    const data = await response.json();
    const status = data.status;

    if (status === 'completed') {
      console.log('视频生成成功！');
      console.log(`视频时长: ${data.actualDuration} 秒`);
      return data.video_url;
    } else if (status === 'failed') {
      console.log(`生成失败: ${data.error}`);
      return null;
    }

    console.log(`当前状态: ${status}，等待中...`);
    await new Promise(resolve => setTimeout(resolve, interval));
  }

  console.log('等待超时');
  return null;
}

// 使用示例
const videoUrl = await waitForVideo('video_fd35ee52-2a98-44a6-b930-29a88ce9b8fd', 'your_api_key');
if (videoUrl) {
  console.log(`视频URL: ${videoUrl}`);
}
